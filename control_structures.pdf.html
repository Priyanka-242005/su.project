Control Structures in Python

1. Conditional Statements

Conditional statements allow you to execute certain code based on conditions. The main statements are if, elif, and else.

if condition:
    # Code to execute if condition is true
elif another_condition:
    # Code to execute if another_condition is true
else:
    # Code to execute if neither condition is true

2. Example of Conditional Statements

Here is a simple example of using conditional statements:

age = 18

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

3. Loops

Python has two types of loops: for and while.

4. For Loop

The for loop is used to iterate over a sequence (like a list or a string).

for i in range(5):
    print(i)

5. While Loop

The while loop runs as long as a specified condition is true.

count = 0
while count < 5:
    print(count)
    count += 1

6. Break and Continue

You can control the flow of loops using break to exit the loop and continue to skip the current iteration.

for i in range(10):
    if i == 5:
        break  # Exit the loop
    print(i)

for i in range(10):
    if i % 2 == 0:
        continue  # Skip even numbers
    print(i)
