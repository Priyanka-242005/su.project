File Handling in Python

1. Opening a File

To work with a file, you first need to open it using the built-in open() function.

file = open('example.txt', 'r')  # Opens a file for reading

2. Reading from a File

There are several methods to read from a file:

content = file.read()  # Reads the entire file
line = file.readline()  # Reads one line
lines = file.readlines()  # Reads all lines into a list

3. Writing to a File

You can write to a file by opening it in write ('w') or append ('a') mode.

file = open('example.txt', 'w')  # Opens a file for writing (overwrites existing content)
file.write("Hello, World!")  # Writes a string to the file

4. Closing a File

Always close a file when you're done to free up system resources.

file.close()  # Closes the file

5. Using 'with' Statement

The with statement is a context manager that automatically closes the file.

with open('example.txt', 'r') as file:
    content = file.read()  # Reads the entire file
# No need to explicitly close the file

6. File Modes

Common file modes include:

  • 'r': Read (default)
  • 'w': Write (overwrites existing content)
  • 'a': Append (adds to the end of the file)
  • 'b': Binary mode (e.g., 'rb' or 'wb')

7. Example of File Handling

Here's a complete example of reading from and writing to a file:

# Writing to a file
with open('example.txt', 'w') as file:
    file.write("Hello, World!\n")
    file.write("This is a test file.\n")

# Reading from the file
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)  # Outputs the content of the file
