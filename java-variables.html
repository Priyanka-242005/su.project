Variables and Data Types

What are Variables?

Variables in Java are containers that store data values. Every variable must be declared with a data type, which determines the size and layout of the variable's memory. A variableâ€™s data type also defines the range of values that can be stored inside it and the operations that can be performed on that data.

Types of Variables

  • Local Variables: Declared inside a method and accessible only within that method.
  • Instance Variables: Declared inside a class but outside any method. They belong to an object and are accessed through the object reference.
  • Static Variables: Declared with the `static` keyword inside a class but outside any method. They belong to the class and are shared by all instances of the class.

Primitive Data Types

Java has 8 primitive data types which are predefined by the language and named by a keyword. These types serve as the building blocks of data manipulation in Java.

  • byte: 8-bit signed integer. Range: -128 to 127.
  • short: 16-bit signed integer. Range: -32,768 to 32,767.
  • int: 32-bit signed integer. Range: -231 to 231-1.
  • long: 64-bit signed integer. Range: -263 to 263-1.
  • float: 32-bit floating-point number.
  • double: 64-bit floating-point number (used for precision).
  • char: 16-bit Unicode character. Range: '\u0000' (0) to '\uffff' (65,535).
  • boolean: Represents one bit of information. Only two possible values: `true` and `false`.

Reference Data Types

In Java, reference types refer to objects. These variables are used to access objects, and they store the memory address of an object. Common reference data types include:

  • Arrays: Containers that hold multiple values of the same data type.
  • Classes: Templates for creating objects. When an object is created, it is referenced by its class type.
  • Interfaces: Similar to classes but they contain abstract methods that must be implemented by a class.

Variable Declaration and Initialization

Variables in Java must be declared before they are used. Here's the syntax for declaring a variable:

data_type variable_name = value;

Example:

int age = 25;

Here, `int` is the data type, `age` is the variable name, and `25` is the value assigned to the variable.

Type Casting

Type casting is converting one data type to another. Java provides two types of casting:

  • Widening (Automatic): Converting a smaller type to a larger type. Example: `int` to `long`.
  • Narrowing (Explicit): Converting a larger type to a smaller type. Example: `double` to `float`. This requires explicit casting.

Example of explicit casting:

double d = 9.78;
int i = (int) d;  // Narrowing casting
