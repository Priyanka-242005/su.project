Functions and Modules in Python

1. Introduction to Functions

A function is a block of code that performs a specific task. Functions can take inputs and return outputs.

def function_name(parameters):
    # Code to execute
    return result

2. Calling a Function

Once a function is defined, you can call it by using its name and passing the required arguments.

result = function_name(arguments)

3. Example of a Function

Here is an example of a simple function that adds two numbers:

def add_numbers(a, b):
    return a + b

sum = add_numbers(5, 3)
print(sum)  # Output: 8

4. Introduction to Modules

Modules are files containing Python code that can define functions, classes, and variables. They allow for code reuse.

import module_name

5. Importing Specific Functions

You can also import specific functions from a module using the following syntax:

from module_name import function_name

6. Example of Importing Modules

Hereâ€™s an example of importing the math module to use the sqrt function:

import math

result = math.sqrt(16)
print(result)  # Output: 4.0

7. Creating Your Own Module

You can create your own module by saving your functions in a .py file and then importing it.

# my_module.py
def greet(name):
    return f"Hello, {name}!"

# main.py
from my_module import greet
print(greet("Alice"))  # Output: Hello, Alice!
