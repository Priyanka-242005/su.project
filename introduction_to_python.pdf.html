Introduction to Python

1. What is Python?

Python is a high-level, interpreted programming language known for its readability and simplicity. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

2. Basic Syntax

Python uses indentation to define code blocks. Here’s a simple example:

if True:
    print("Hello, World!")

3. Data Types

Python has several built-in data types:

  • Integer: Whole numbers (e.g., 10, -5)
  • Float: Decimal numbers (e.g., 10.5, -3.14)
  • String: Text (e.g., "Hello, World!")
  • Boolean: Represents True or False values

4. Variables

Variables in Python are created when you assign a value to them. Here’s how to create a variable:

x = 5
name = "Alice"

5. Comments

Comments are used to explain the code and are ignored by the interpreter. In Python, comments start with a #:

# This is a comment
print("Hello, World!")  # This prints a message
