Data Structures in Python

1. Lists

Lists are ordered, mutable collections that can hold mixed data types.

my_list = [1, 2, 3, 'apple', 'banana']
my_list.append('orange')  # Adds 'orange' to the end
print(my_list[0])  # Output: 1

2. Tuples

Tuples are ordered, immutable collections. They can also hold mixed data types.

my_tuple = (1, 2, 3, 'apple', 'banana')
print(my_tuple[1])  # Output: 2

3. Sets

Sets are unordered collections of unique elements. They are mutable.

my_set = {1, 2, 3, 4, 5}
my_set.add(6)  # Adds 6 to the set
print(my_set)  # Output: {1, 2, 3, 4, 5, 6}

4. Dictionaries

Dictionaries are unordered collections of key-value pairs. They are mutable.

my_dict = {'name': 'Alice', 'age': 25}
print(my_dict['name'])  # Output: Alice
my_dict['age'] = 26  # Update age

5. List Comprehensions

A concise way to create lists using a single line of code.

squared = [x**2 for x in range(10)]
print(squared)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

6. Using Tuples for Unpacking

Tuples can be used to unpack values into multiple variables.

point = (3, 5)
x, y = point
print(x, y)  # Output: 3 5

7. Nesting Data Structures

Data structures can be nested inside one another.

nested_list = [[1, 2], [3, 4], [5, 6]]
print(nested_list[1][0])  # Output: 3
